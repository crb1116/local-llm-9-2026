## 🔑 Troubleshooting Common Issues

*   **The app refuses to open**: Restart your computer first. Ensure your antivirus software does not block the application. You might need to add an exception for the installation folder.
*   **Responses run very slowly**: Close other heavy applications like video games or browsers with many tabs. If your computer lacks a dedicated graphics card, the model takes longer to generate text.
*   **The browser shows an error**: Confirm the console window remains open. If the window hides, the server stops working. Refresh the page in your browser.
*   **Disk space warnings**: These AI models require significant storage space. Check the properties of your installation folder to see how much room you occupy. Delete models you no longer use to recover space.
*   **Update cycles**: Check the download page every month. We update the engine to include better versions of LLMs as they become available.
